# Шахматы

## Что реализовано

- `Board` хранит доску, текущий ход, историю и специальные состояния партии.
- `Move` описывает один ход как объект: старт, финиш, взятие, рокировка, взятие на проходе, превращение.
- Все фигуры оформлены как классы-наследники `Piece`.
- Каждая фигура сама вычисляет свои ходы через метод `pseudo_legal_moves`.
- Доска фильтрует ходы и оставляет только легальные, то есть не оставляющие своего короля под шахом.

## Дополнительные задания

Суммарная сложность: `5`

- Новые виды фигур (`1`)
  - `Chancellor` = ладья + конь
  - `Archbishop` = слон + конь
  - `Sentinel` = ходы короля на дистанцию до 2 клеток + ходы коня
- Откат ходов (`1`)
- Подсказка допустимых ходов (`1`)
- Подсказка угрожаемых фигур и выделение шаха (`1`)
- Сложные правила пешек: взятие на проходе и превращение (`1`)
